Setting up FireFox

This article explains how to setup FireFox for testing with WatiN.

  • Make sure you have Firefox version 2.0 or higher installed.
  • Next you should download the most recent release of WatiN and unzip it into an appropriate directory. Assuming you have unzipped the release to c:\Program Files\WatiN\
  • A plugin jssh is used by WatiN to automate Firefox to carry out tests. To install the plugin:
    1. Open FireFox
    2. Click File -> Open File
    3. Navigate to your WatiN installation folder i.e. c:\Program Files\WatiN\Mozilla
    4. Now pick the right version of jssh-....xpi. This depends on the FireFox version you have installed.
