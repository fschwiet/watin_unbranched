Below is the roadmap for future releases of WatiN. The releases are both feature and time driven. Every new release needs at least one or two major new features before it will be released. The release cycle will be 2 month. This gives me:

-          2 weeks to sit back and enjoy the download count of the previous release J.

-          4 weeks to do the programming.

-          and 2 weeks to make it release ready.

 

If you have a feature request thats not on this roadmap please submit it through the feature request tracker on SourceForge.

 

Disclaimer: This roadmap states my intentions but is not meant to rule out any changes in planned features, release dates or what so ever.

 

Current release:

WatiN 1.0.0 was released on March 1, 2006. Read the release notes here and/or download it here.
